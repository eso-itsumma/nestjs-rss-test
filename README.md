
## Описание

Репозиторий для выполнения тестового задания

## Установка

```bash
$ yarn install
```

## Запуск приложения

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Тесты

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
